# 🧠 کنکور ارشد روانشناسی — PWA

اپلیکیشن ۳۰۰ سوال پرتکرار کنکور ارشد روانشناسی به صورت Progressive Web App

## ساختار فایل‌ها

```
/
├── index.html       ← صفحه اصلی
├── style.css        ← استایل‌ها
├── app.js           ← منطق برنامه + سوالات
├── manifest.json    ← تنظیمات PWA
├── sw.js            ← Service Worker (آفلاین)
└── icons/
    ├── icon-48.png
    ├── icon-72.png
    ├── icon-96.png
    ├── icon-144.png
    ├── icon-192.png
    └── icon-512.png
```

## نحوه انتشار روی GitHub Pages

1. یک ریپو جدید در GitHub بسازید
2. همه فایل‌ها را آپلود کنید
3. به Settings → Pages بروید
4. در Source گزینه **main branch / root** را انتخاب کنید
5. Save کنید — آدرس سایت شما آماده می‌شود

## تبدیل به اپ Android با PWABuilder

1. به [pwabuilder.com](https://pwabuilder.com) بروید
2. آدرس GitHub Pages خود را وارد کنید
3. روی Start کلیک کنید
4. گزینه Android را انتخاب و دانلود کنید
